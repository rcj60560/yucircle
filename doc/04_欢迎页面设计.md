# 羽圈 App - 欢迎页面设计 (Welcome/Onboarding Page)

> **目的**：首次启动应用时，向用户展示应用核心价值和社区规则  
> **位置**：登录后 → 进入 MainPage 前的可选页面  
> **实现方式**：可选的欢迎页卡片轮播 + "我已阅读"按钮

---

## 实现建议

### 技术方案

**方案 A（推荐）**：独立的 WelcomePage
```dart
// lib/pages/welcome/welcome_page.dart
class WelcomePage extends StatefulWidget {
  // PageView 轮播卡片
  // 支持左右滑动或"下一步"按钮翻页
  // 最后一页勾选"不再显示"后，设置 SharedPreferences 标志
}

// lib/pages/splash/splash_page.dart 中判断是否显示 Welcome 页
if (isFirstTime) {
  Get.offNamed('/welcome');  // 先显示欢迎页
} else {
  Get.offNamed('/main');     // 直接进入主应用
}
```

**方案 B**：集成到 SplashPage
```dart
// 启动画面后直接展示欢迎卡片
// 比较简洁，但功能不如独立页面灵活
```

### UI 设计参考

**多邻国风格元素**：
- ✅ 绿色主色（#4CD964 或类似）
- ✅ 圆角卡片（12-16 dp）
- ✅ 柔和阴影
- ✅ 大字号标题

## 页面流程图

```
SplashPage (启动画面)
    ↓
[检查是否首次启动]
    ├─ 是首次 → WelcomePage (6 张卡片轮播)
    │              ↓
    │         [用户阅读完成]
    │              ↓
    │         [保存 "已查看" 标志]
    │              ↓
    │         进入 MainPage
    │
    └─ 非首次 → 直接进入 MainPage
```
