### 1. 克隆或切换到项目目录

```bash
cd d:\Users\luocj\tf\yucircle\circle
```

### 2. 初始化 Flutter 环境

```bash
flutter clean
flutter pub get
flutter doctor -v
```

### 3. 按照 [01 - Flutter 开发指南] 第一部分创建项目文件结构

### 4. 运行项目

```bash
flutter run
```

- 先查 [02 - 技术栈与依赖] 的配置问题部分
