## 依赖说明

### 核心依赖

| 依赖 | 版本 | 用途 | 为什么选择 |
|------|------|------|----------|
| **get** | 4.6.6 | 状态管理、路由 | 学习成本低、性能好、社区活跃 |
| **dio** | 5.3.1 | HTTP 请求 | 功能完整、易用、拦截器强大 |
| **shared_preferences** | 2.2.2 | 本地存储 | 官方推荐、跨平台支持 |
| **image_picker** | 1.0.4 | 图片选择 | 官方插件、易用 |
| **cached_network_image** | 3.3.1 | 图片缓存 | 自动缓存、内存管理好 |
| **flutter_animate** | 4.1.1 | 动画库 | 简单易用、适合轻量动画 |
| **intl** | 0.19.0 | 国际化 | 官方推荐、日期时间格式 |

### 为什么不选择的库

- ❌ **Provider** - 相比 GetX 功能少，样板代码多
- ❌ **BLoC** - 过于复杂，不适合 MVP 阶段
- ❌ **http** - 功能不如 Dio 完整
- ❌ **GetStorage** - 相比 SharedPreferences 优势不明显
- ❌ **Riverpod** - 学习曲线陡，不适合快速开发

## 安装步骤

### 1. 初始化项目

```bash
# 创建新项目
flutter create --org com.badminton.app yucircle

# 进入项目目录
cd yucircle
```

### 2. 添加依赖

```bash
# 方式一：编辑 pubspec.yaml 后运行
flutter pub get

# 方式二：直接添加依赖
flutter pub add get
flutter pub add dio
flutter pub add shared_preferences
flutter pub add image_picker
flutter pub add cached_network_image
flutter pub add flutter_animate
flutter pub add intl
```

### 3. 验证安装

```bash
# 检查依赖
flutter pub get

# 检查环境
flutter doctor -v

# 尝试运行
flutter run
```

## 常见配置问题

### Q: Android 构建失败，提示 SDK 版本过低

```bash
# 更新 SDK
flutter pub get
flutter pub upgrade

# 或者修改 android/build.gradle
# 调整 compileSdkVersion 和 targetSdkVersion
```

### Q: iOS 构建失败，提示 Pod 不兼容

```bash
# 清理 Pod 缓存
cd ios
rm -rf Pods
rm Podfile.lock
cd ..

# 重新安装
flutter pub get
```

### Q: 图片缓存不工作

确保在 main.dart 中配置了缓存管理器：

```dart
import 'package:cached_network_image/cached_network_image.dart';

// CachedNetworkImage 会自动处理缓存
```

### Q: 本地存储数据丢失

SharedPreferences 在以下情况下数据会丢失：
- 应用卸载
- 用户清除应用数据
- 设备恢复出厂设置

如需持久化，考虑使用 SQLite 或 Hive。

## 性能优化建议

### 1. 内存优化
- 使用 cached_network_image 管理图片缓存
- 在 ListView 中使用 repaint boundaries
- 及时销毁大对象的引用

### 2. 网络优化
- 配置 Dio 的请求超时时间
- 实现请求缓存机制
- 使用 gzip 压缩

### 3. UI 优化
- 避免在 build 方法中做复杂计算
- 使用 const 修饰 Widget
- 用 SingleChildScrollView 替代嵌套 Column

### 4. 构建优化
- 使用 --profile 或 --release 构建
- 启用 Code obfuscation（混淆）
- 分析包大小（flutter analyze）

## 依赖更新策略

### 定期更新
```bash
# 查看可更新的依赖
flutter pub outdated

# 更新所有依赖到最新版本
flutter pub upgrade

# 更新到主版本
flutter pub upgrade --major-versions
```

### 安全更新
- 测试环境先更新，观察是否有问题
- 更新前备份 pubspec.lock
- 关键依赖手动指定版本
